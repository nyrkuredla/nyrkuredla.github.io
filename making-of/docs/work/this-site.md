# Project Overview: Portfolio Site

I created a technical writing portfolio project (this page!) using a docs-as-code approach. Here's what I accomplished and how:

## Project Outcomes

- Demonstrates docs-as-code principles
- Shows Git proficiency through multi-repository management
- Includes a live site deployed through GitHub Pages
- Establishes a foundation for adding AI integration and CI/CD pipelines
- Provides a platform for documenting my learning journey

## Technologies & Tools Used
- **Generative AI (Claude and ChatGPT)**: Ideating, debugging, generating most of this summary, debugging rant receptacle 
- **MkDocs with Material theme**: Documentation framework for creating the site
- **Git**: Version control for the project
- **GitLab**: Primary repository for demonstrating Git workflow
- **GitHub Pages**: Deployment platform for the live documentation site
- **Virtual Environment**: For Python dependency management
- **Bash/PowerShell**: Created aliases for streamlining the workflow

## Project Implementation
1. **Repository Setup**: Created repositories on both GitLab and GitHub
2. **Documentation Structure**: Implemented a MkDocs site with sections for learning journey, technical guides, and style guides
3. **Deployment Process**: Configured GitHub Pages to host the documentation site
4. **Workflow Automation**: Created scripts/aliases to simplify pushing to multiple repositories

## Challenges Defeated
- Resolved Git branch conflicts between main/master
- Fixed GitHub Pages deployment issues
- Established connections between multiple Git remotes
- Troubleshooted* site rendering problems

------
*Author's note*: My brain has trouble deciding whether *troubleshooted* or *troubleshot* sounds better here (or rather worse). This is probably due to *troubleshoot* being a neologism. Per morphology class at [grad school](https://sites.google.com/view/uwr-ethel/programme-description), neologistic verbs default to regular (i.e., *-ed*) endings; so far, so good, but the problem here is that *troubleshoot* includes *shoot* which has an irregular past participle (*shot*). Basically, there's no definitive right answer and they both manage to sound weird and vaguely upsetting. One of those English quirks that makes one pull one's hair out as an ESL teacher.
